# 🚆 BTS Skytrain Passenger Analysis (2000-2025)

## 📌 Overview
This project explores and analyzes the passenger traffic data of the Bangkok Mass Transit System (BTS Skytrain) over a 25-year period, from 2000 to 2025. The goal of this analysis is to uncover ridership trends, understand seasonal behaviors, and evaluate the impact of major historical events on public transportation usage.

## 🎯 Business Questions & Objectives
Through this Exploratory Data Analysis (EDA), we aim to answer the following key questions:
* **Long-Term Trends:** How has the overall ridership grown or declined from 2000 to 2025?
* **Impact Analysis:** How did significant events (e.g., the COVID-19 pandemic) affect passenger volume, and how long did the recovery take?
* **Seasonality:** Are there specific months or seasons with consistently higher or lower passenger traffic?
* **Future Projections:** Based on historical data, what are the expected ridership trends for the upcoming years?

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Libraries:** `pandas` (Data Manipulation), `matplotlib` & `seaborn` (Data Visualization), `numpy` (Numerical Computations)
